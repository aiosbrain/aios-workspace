## Share a skill

Push a skill to the Team Brain so teammates can pull it:

```bash
aios push skill <name>      # share SKILL.md + its reference files
aios pull skill <name>      # fetch a shared skill into 1-inbox/from-brain/skills/
aios install-skill <name>   # promote a pulled skill into .claude/skills/ (explicit, append-only)
```

Pulled skills are executable code — install is always a deliberate act; they never auto-activate on pull.
